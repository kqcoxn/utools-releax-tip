# utools-relax-remainder

## 这是什么？

「休息提醒」是一个基于 uTools 的插件，可以帮助你定时休息。

## 主要功能

- **定时提醒**：每间隔一段自定义时间，插件会提醒你休息一下，可设置稍后提醒与强制出现。
- **休息计时**：休息结束后，插件会提醒你休息结束。
- **使用计时**：插件会记录你每天的使用时间与本次使用时间，并显示在插件的主界面上。
- **一言**：插件会随机获取一句名言，并显示在插件的主界面上。

## What's Different?

目前 uTools 已经有了休息提醒的插件，但没找到有插件可以做到稍后提醒与强制出现功能，所以为了这碟醋包了这盘饺子 QWQ
